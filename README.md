## 已确认条件

- 课程共 6 天，每天 2 个课时，每课时 50 分钟，总计 600 分钟。
- 学生是高中毕业、即将进入大学的准大一新生，大多将学习理工科；课程按零编程基础设计。
- 课程需要覆盖 Python 基础语法、判断、循环、函数、数据容器、对象可变性和深浅拷贝。
- 第 0 节为课程介绍与编程概述；编程思想贯穿后续全部内容。
- 课表只规定对外可见的整体覆盖范围，不锁定实际授课顺序、单项时长或实践组织方式。
