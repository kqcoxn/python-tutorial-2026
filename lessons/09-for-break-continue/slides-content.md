## 第 11 页 · B · `break` 到哪里

**屏显文字：**

>
> 它不会关闭 Python 程序。

## 第 12 页 · B · `continue`

**屏显文字：**

> ## 跳过本轮剩余代码，进入下一轮
>
> ```python
> for number in range(1, 8):
>     if number % 2 == 0:
>         continue
>     print(number)
> ```
>
> 输出：1、3、5、7

## 第 14 页 · B · `while` 中的 `continue` 风险（点击揭示）

**屏显文字：**

> ## 必要更新被跳过，就会死循环
>
> ```python
> count = 0
> while count < 5:
>     if count == 2:
>         continue
>     count += 1
> ```
>
> **揭示：到 2 后，`count += 1` 永远执行不到。**

---

## 第 15 页 · B · 嵌套循环中的 `break`

**屏显文字：**

> ## 只退出内层，外层仍继续
>
> ```python
> for row in range(1, 4):
>     for column in range(1, 5):
>         if column == 3:
>             break
>         print(row, column)
> ```
>
> 每一行只输出第 1、2 列。
