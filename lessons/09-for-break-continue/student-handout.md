## 4. `break`：提前结束循环

```python
secret = "python"

for attempt in range(1, 4):
    answer = input(f"第 {attempt} 次尝试：")

    if answer == secret:
        print("回答正确")
        break

    print("回答错误")
```

| 输入情况 | 循环轮数 |
| --- | ---: |
| 第一次正确 | |
| 第三次正确 | |
| 三次都错 | |

`break` 只结束最近一层循环，循环后的代码仍会继续。

## 5. `continue`：跳过本轮

```python
for number in range(1, 8):
    if number % 2 == 0:
        continue

    print(number)
```

对比：

- `break`：不再循环。
- `continue`：本轮剩余代码不执行，进入下一轮。

## 11. 快速排错

| 现象 | 检查 |
| --- | --- |
| 少输出结束值 | `range()` 终点不包含 |
| 范围为空 | 步长方向是否匹配起止值 |
| `break` 后报告仍输出 | 正常，它只结束最近循环 |
| `continue` 后仍扣资源 | 跳转是否发生在结算前 |
| `while` 停在同一值 | 更新是否被 `continue` 跳过 |
| 嵌套 `break` 未退出外层 | 它只退出最近一层 |
