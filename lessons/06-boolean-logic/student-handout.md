## 2. 比较运算

```python
score = 60

print(score == 60)
print(score != 60)
print(score > 60)
print(score < 60)
print(score >= 60)
print(score <= 60)
```

`=` 表示赋值，`==` 才表示比较是否相等。

## 7. 真值表与短路

| A | B | `A and B` | `A or B` |
| --- | --- | --- | --- |
| `True` | `True` | | |
| `True` | `False` | | |
| `False` | `True` | | |
| `False` | `False` | | |

短路方向：

- `and` 左边为 `False` 时，结果已经确定为 `False`。
- `or` 左边为 `True` 时，结果已经确定为 `True`。

这意味着后半部分可能不再计算。本阶段不要依赖复杂副作用使用短路。

## 10. 快速排错

| 现象 | 检查 |
| --- | --- |
| 条件中写 `=` | 相等比较应使用 `==` |
| `"True"` 类型不对 | 去掉引号才是布尔值 |
| 边界值失败 | `>` 与 `>=` 是否选对 |
| 组合条件相反 | 用“全部”或“至少一个”重新翻译 |
| `not` 看不清范围 | 加括号或保存中间变量 |
| 字符串不匹配 | 大小写和空格 |

## 11. 本章速查表

| 目标 | 写法 |
| --- | --- |
| 相等 / 不相等 | `==` / `!=` |
| 大于、小于 | `>` / `<` |
| 至少、至多 | `>=` / `<=` |
| 全部成立 | `A and B` |
| 至少一个成立 | `A or B` |
| 反转 | `not A` |
