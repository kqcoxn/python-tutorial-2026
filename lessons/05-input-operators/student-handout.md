## 10. 安全与排错

不要使用 `eval(input())`。它会执行输入内容，不是安全的数字转换方式。

| 现象 | 检查 |
| --- | --- |
| `"2" + "3"` 得到 `"23"` | 是否转换为数字 |
| 字符串不能相减 | `input()` 是否仍为 `str` |
| `ValueError` | 输入是否符合整数或小数格式 |
| `/` 与 `//` 结果不同 | 需要普通除法还是整除 |
| 数值突然变成负数 | 是否写成 `=-` 而不是 `-=` |
| 结果顺序错误 | 是否需要括号 |
