## 6. 第一次使用断点

```python
print("第 1 步：准备")
print("第 2 步：执行")
print("第 3 步：完成")
```

操作检查：

- [ ] 在第 2 行左侧单击，看到断点标记。
- [ ] 使用 Debug 运行。
- [ ] 程序在第 2 行执行前暂停。
- [ ] 使用 Step Over 逐行执行。
- [ ] 使用 Resume 继续运行。
- [ ] 移除断点。

断点表示程序暂停供我们观察，不表示这一行有错误。

## 9. 快速排错

| 现象 | 先检查 |
| --- | --- |
| 注释显示在输出中 | `#` 是否写进了引号 |
| 代码变灰且不执行 | 行首是否误加 `#` |
| `SyntaxError` | 引号、括号及上一行 |
| `NameError` | 名称拼写和大小写 |
| `unexpected indent` | 行首是否有多余空格 |
| Debug 直接结束 | 断点、当前文件和运行配置 |
| 程序停住 | 是否正停在断点 |
