## 3. 从布尔值到执行路径

条件表达式负责回答“是或否”，条件分支负责决定“接下来执行什么”。

```text
检查条件
├─ True  -> 执行这个分支的代码
└─ False -> 跳过这个分支，检查后续路径
```

### 3.1 单个 `if`

```python
if 条件:
    条件为 True 时执行的语句

继续执行的语句
```

注意：

- 条件末尾是英文冒号 `:`。
- 分支内的代码要缩进，通常使用 4 个空格。
- 取消缩进表示该代码块已经结束。
- 单个 `if` 不一定需要 `else`。

示例：

```python
score = int(input("请输入分数："))

if score >= 60:
    print("及格")

print("判定结束")
```

### 3.2 `if-else`

当两种情况必须选择一种时使用：

```python
if 条件:
    条件为 True 时执行
else:
    条件为 False 时执行
```

示例：

```python
number = int(input("请输入一个整数："))

if number % 2 == 0:
    print("这是偶数。")
else:
    print("这是奇数。")
```

### 3.3 `if-elif-else`

当多个选项中一次只执行一个时使用：

```python
if 条件 1:
    条件 1 最先成立时执行
elif 条件 2:
    条件 1 不成立、条件 2 成立时执行
else:
    前面的条件都不成立时执行
```

Python 从上到下检查条件，遇到第一个成立的条件后执行对应代码块，并跳过这条分支链剩余的部分。

## 5. 多个 `if` 还是 `if-elif-else`

先问：这些情况能不能同时发生？

- 能同时发生：通常使用多个独立 `if`。
- 一次只能选一个：通常使用一条 `if-elif-else` 分支链。

## 6. 嵌套判断：周末安排

这个轻松示例有两个问题：

1. 今天是否为周末？
2. 如果是周末，作业是否已经完成？

第二个判断只在第一个问题的答案为“是”时发生，因此可以写在外层分支内部。

```python
is_weekend = input("今天是周末吗（是/否）：")

if is_weekend == "是":
    homework_done = input("作业写完了吗（是/否）：")

    if homework_done == "是":
        print("可以安心打游戏。")
    else:
        print("先写作业，再打游戏。")
else:
    print("先去上课。")
```

执行路径：

```text
今天是否为周末？
├─ 否 -> 先去上课
└─ 是 -> 作业是否写完？
         ├─ 是 -> 可以安心打游戏
         └─ 否 -> 先写作业，再打游戏
```

## 12. 快速排错

| 现象 | 优先检查 |
| --- | --- |
| 条件行出现 `SyntaxError` | 行末是否有英文冒号 `:` |
| `elif` 或 `else` 报错 | 是否与所属 `if` 对齐 |
| 输入 `1` 却不进入任务 1 | 是否写成与字符串 `"1"` 比较 |
| 出现 `if choice = "1"` 错误 | 条件比较应使用 `==` |
| 分支只执行了第一条更新 | 同一分支内的语句是否保持相同缩进 |
| 一个选择触发多个结果 | 互斥选择是否误用了多个独立 `if` |
| 90 分进入 60 分分支 | 更严格的条件是否放在前面 |
| 无效输入仍扣除资源 | 日常消耗是否受 `task_completed` 控制 |
| 任务 2 的资源超过 100 | 是否分别处理三项资源上限 |

排错顺序：

1. 找到第一条红色错误信息和对应行。
2. 说明你期望程序进入哪个分支。
3. 检查条件值、比较对象和缩进层级。
4. 每次只修改一个原因，再重新运行同一组输入。

## 13. 本章速查表

| 目标 | 结构 |
| --- | --- |
| 条件成立时才执行 | `if 条件:` |
| 两种情况选一种 | `if ... else ...` |
| 多个互斥选项选一个 | `if ... elif ... else ...` |
| 多个情况可以同时发生 | 多个独立 `if` |
| 在一个分支内再判断 | 嵌套 `if` |
| 比较是否相等 | `==` |
| 表示代码块 | 统一缩进，通常 4 个空格 |

常用代码形状：

```python
if condition:
    action
```

```python
if condition:
    action_a
else:
    action_b
```

```python
if condition_a:
    action_a
elif condition_b:
    action_b
else:
    action_c
```
