# Python 基础课程章节材料

每章均包含教师教案和学生讲义。普通语法演示使用传统或轻量示例，火星基地项目集中放在章节后半段的项目增量、练习与验收中。

根据 [`docs/课程安排.xlsx`](../docs/课程安排.xlsx)，完整课程共 6 天、12 个课时，每天 2 课时，每课时 50 分钟。当前目录的第 1–10 章是覆盖前 3 天、第 1–6 课时的知识模块，不与课时一一对应：第 1 课时使用课程导入和第 1 章，第 2–3 课时推进第 2–6 章，第 4 课时使用第 7 章，第 5 课时使用第 8–9 章，第 6 课时使用第 10 章完成项目实践（一）。第 7–12 课时材料待后续建设。

## 使用顺序

1. 教师先阅读总课程设计和本章教师教案。
2. 课堂前向学生发放对应学生讲义和上一项目版本。
3. 先完成普通语法演示，再明确切换到项目集中编写。
4. 使用章节检查点决定继续、补练或暂停。
5. 最终参考实现只供教师验证，不提前发给学生。
