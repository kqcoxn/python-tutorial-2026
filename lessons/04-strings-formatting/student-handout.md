## 2. 字符串与引号

```python
line_a = "Python says: 'Hello!'"
line_b = '老师说："先预测，再运行。"'

print(line_a)
print(line_b)
```

规则：外层使用单引号时，内容中可以直接出现双引号；反过来也可以。代码引号必须使用英文半角 `'` 或 `"`。

## 5. 转义字符

```python
print("第一行\n第二行")
print("姓名\t分数")
print("C:\\Users\\student")
print("他说：\"Hello!\"")
```

| 写法 | 效果 |
| --- | --- |
| `\n` | 换行 |
| `\t` | 制表间隔 |
| `\\` | 输出一个反斜杠 |
| `\"` | 输出双引号 |
| `\'` | 输出单引号 |

`\t` 只提供制表间隔，不保证任何长度的文字都能形成整齐表格。

## 9. 快速排错

| 现象 | 检查 |
| --- | --- |
| 输出 `{name}` | 字符串前是否有 `f` |
| 引号中间报 `SyntaxError` | 内外引号是否冲突或缺失 |
| 字符串与整数拼接报错 | 改用 f-string |
| 没有换行 | 是否写成正确的 `\n` |
| 路径报转义错误 | 每个反斜杠是否写成 `\\` |
| 花括号报错 | `{` 和 `}` 是否成对 |
