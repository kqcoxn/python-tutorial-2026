## 1. 章节定位

本章让学生把分散的变量组织成可读输出。重点是字符串边界、f-string 和常用转义字符；不追求字符串方法大全。前半段使用姓名、成绩、台词和路径等普通示例，后半段集中制作 V3 状态面板。

## 2. 学习目标与验收证据

| 学习目标 | 验收证据 |
| --- | --- |
| 使用单引号和双引号创建字符串 | 能选择外层引号避免不必要转义 |
| 用 `+` 拼接字符串 | 能解释拼接结果和空格来源 |
| 使用 f-string 插入变量 | 输出中显示变量值而不是 `{变量名}` |
| 使用常见转义字符 | 能正确输出换行、制表、反斜杠和引号 |
| 修复字符串相关错误 | 能处理缺引号、缺 `f` 和字符串数值拼接错误 |
| 完成 V3 状态面板 | 基地、任务日、三项资源和积分排版正确 |

最低标准：学生能用 f-string 输出至少三个变量，并正确使用 `\n` 生成多行文字。

## 4. 核心词汇

| 词汇 | 含义 |
| --- | --- |
| 字符串 | 由引号包围的文本值 |
| 拼接 | 把多个字符串连接成一个字符串 |
| f-string | 在字符串中直接插入表达式值的格式化写法 |
| 占位表达式 | f-string 中 `{}` 内要计算并显示的内容 |
| 转义字符 | 以反斜杠开头、表示特殊字符的写法 |

## 6. 核心教学步骤

### 步骤 1：选择字符串外层引号

```python
line_a = "Python says: 'Hello!'"
line_b = '老师说："先预测，再运行。"'

print(line_a)
print(line_b)
```

单引号和双引号都能创建字符串。外层使用一种，引号内容中可直接出现另一种。不要把中文弯引号 `“”` 当作代码引号。

### 步骤 2：字符串拼接

```python
first_name = "Lin"
last_name = "Mu"

print(first_name + " " + last_name)
```

追问空格来自哪里。强调 `+` 拼接不会自动增加空格；拼接的两侧都必须是字符串。

### 步骤 3：使用 f-string

```python
name = "小林"
score = 96

print(f"{name} 本次得分：{score}")
```

执行前先识别：前缀 `f`、字符串边界、两个 `{}` 占位表达式。对比：

```python
print("{name} 本次得分：{score}")
```

没有 `f` 时花括号只是普通文字。

### 步骤 4：常用转义字符

```python
print("第一行\n第二行")
print("姓名\t分数")
print("C:\\Users\\student")
print("他说：\"Hello!\"")
```

| 写法 | 输出效果 |
| --- | --- |
| `\n` | 换行 |
| `\t` | 制表间隔 |
| `\\` | 一个反斜杠 |
| `\"` | 双引号字符 |
| `\'` | 单引号字符 |

说明 `\t` 不是严格表格工具，不同内容长度可能导致视觉错位；本阶段只用于认识转义。

### 步骤 5：组合变量、f-string 和换行

```python
name = "小林"
score = 96
rank = 2

print(f"姓名：{name}\n分数：{score}\n名次：{rank}")
```

验证要求：输出三行，变量值正确，不显示花括号。

## 7. 常见误解与排查

| 现象 | 原因 | 处理 |
| --- | --- | --- |
| 输出 `{name}` | f-string 前遗漏 `f` | 改为 `f"..."` |
| `SyntaxError` 指向文字中间 | 内外引号冲突或缺失 | 标出字符串开始与结束，再选择另一种外层引号 |
| `TypeError`：字符串和整数不能拼接 | 用 `+` 直接连接 `str` 与 `int` | 优先改用 f-string |
| 输出出现字面量 `\n` | 写成了 `\\n` 或转义需求理解错误 | 根据目标决定换行还是显示反斜杠 |
| Windows 路径触发转义错误 | 反斜杠未转义 | 将每个 `\` 写成 `\\` |
| f-string 花括号报错 | 左右花括号不成对 | 检查每个 `{` 对应的 `}` |
