4. 风格标记：S = 瑞士纸白（叙事/过渡）、B = 终端绿（知识/代码）、C = 像素轨道（案例）。

## 第 11 页 · B · 字符串与引号

**屏显文字：**

> ## 字符串：文本必须穿"引号外套"
>
> ```python
> print("abc")      # 字符串：用引号包围
> print('组一辈子的乐队')  # 单引号也可以
> print(123)        # 数字不用引号
> print(3.6)
> ```
>
> **字符串**（string）：任意数量字符组成的文本。
> 引号 = 告诉解释器"这是文字，不是指令"。

**互动说明（不上屏）**：先问学生"print(123) 为什么不加引号？"再点击揭示运行结果。
