## 3. 循环四部分

```text
初始状态 -> 检查条件 -> 执行循环体 -> 更新状态 -> 回到条件
```

在下面代码中标出初值、条件、循环体和更新：

```python
remaining = 3

while remaining > 0:
    print(remaining)
    remaining -= 1

print("开始！")
```

若把 `-=` 写成 `+=`，程序会：正常结束 / 可能死循环。

## 10. 快速排错

| 现象 | 检查 |
| --- | --- |
| 重复同一个值 | 是否更新循环变量 |
| 越来越远离结束 | 更新方向是否错误 |
| 一次都不运行 | 初始条件是否为 `False` |
| 少一次或多一次 | `<` 与 `<=` |
| 输入验证不结束 | `and` 条件和重新输入位置 |
| 网格只完成一行 | 内层计数器是否每轮重置 |
| 无效任务消耗天数 | 结算代码缩进与位置 |
