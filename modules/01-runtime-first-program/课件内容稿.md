## 1. 内容与呈现原则

- 每页的“屏幕内容”是学生当场需要看到的文字；“制作备注”不进入最终画面。
- 问题页与答案页分开，确保学生先预测或动手，再看到结果。
- 每页风格必须是四主题之一（瑞士纸白 / 终端绿 / 账本蓝 / 星图青），以页为最小单位整体切换；不写“过渡”“呼应两套视觉”“以某色为强调色”等混用或叠加描述。本节使用瑞士纸白（概念关系、操作步骤、切换与总结）和终端绿（代码与运行结果）。
- M1 只建立环境确认和最小执行操作，不正式讲解 Python 语法，不展开 Traceback。
- 所有代码使用英文半角引号；故障示例（全角引号、未保存）必须明确标记为“故意演示”。
