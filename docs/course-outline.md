# Python 课程总课纲与课时安排

> 排课来源：[`课程安排.xlsx`](课程安排.xlsx)（2026-07-21 同步）。
> 补充条件：课时结构与学生画像由教师于 2026-07-21 确认。
> 说明：每课时 50 分钟，每天连续安排 2 个课时；课程共 6 天、12 个课时，总计 600 分钟（10 小时）。
> 教学材料：当前仓库中的 `lessons/01`–`lessons/10` 覆盖第 1–6 课时，后续课时材料将按本安排继续建设。

## 教学对象

- 学生已高中毕业，是即将进入大学的准大一新生。
- 课程语境中的“清北苗子”，指学生即将就读清华、北大层次的高校，不是指其当前所在高中或年级。
- 学生整体学习能力和抽象推理能力较强；教学材料仍按 Python 零基础设计，不跳过首次编程所需的环境操作、语法示范和调试训练。

## 课程团队

- 主讲：马锦艺（第 1–12 课时）
- 助教：梁健、谢雨荷，按课时分工协作

## 内容结构

| 课时范围 | 内容板块 | 阶段成果 |
| --- | --- | --- |
| 1–5 | 环境、基础语法、判断与循环 | 能够独立编写、运行和调试包含输入、分支与循环的 Python 程序 |
| 6 | 项目实践（一） | 综合运用前 5 课时知识完成第一个阶段项目 |
| 7 | 函数 | 能够使用函数组织代码，理解参数、返回值与作用域 |
| 8 | 项目实践（二） | 使用函数重组并扩展阶段项目 |
| 9–11 | 数据容器与对象可变性 | 掌握可变、不可变数据容器以及赋值与拷贝的行为差异 |
| 12 | 项目实践（三） | 综合运用函数、数据容器与对象模型完成课程项目 |

## 备课约束

- 第 1–6 课时继续使用“火星基地资源调度模拟器”作为贯穿项目，并在第 6 课时完成项目实践（一）。
- 第 7 课时才正式引入自定义函数、形参实参、返回值和作用域；此前材料不得依赖这些知识。
- 第 9 课时开始正式引入可变数据容器，第 10 课时讲授不可变数据容器，第 11 课时集中处理深浅拷贝与对象可变性。
- 第 6、8、12 课时以项目实践为主，应优先保证学生独立编写、调试、解释和展示的时间。
- 若电子表格与其他课程说明出现冲突，以 `课程安排.xlsx` 的课时、课程内容和人员安排为准。
