# Python 基础与 AI 编程实践 · 课程总课纲

> 来源：主讲教师提供（2026-07-19）。
> 说明：本课纲**按章节划分，不按课时划分**；课堂按章节检查点推进，不绑定具体课时。
> 配套：第一阶段（第 1–11 章）的教学设计见 `python-foundations-course-design.md`，其中 §5 将其重组为 10 个课件章节（对应 `lessons/01`–`lessons/10`），个别知识点在相邻章节间微调（数值类型并入课件第 3 章、转义字符并入课件第 4 章）。

## 阶段划分（名称暂定）

| 阶段 | 章节 | 内容 | 主讲 |
| --- | --- | --- | --- |
| 一 · Python 基础 | 1–11 | 环境、输出、变量、判断、循环、课上作业 | 本课程主讲教师 |
| 二 · Python 进阶 | 12–20 | 字符串、列表、元组字典集合、类型转换、深浅拷贝、函数 | 待定 |
| 三 · AI 编程实践 | 21–30 | LLM 与 Agent、利用 Agent 编程、AI 方法构建 | 待定 |

## 完整课纲（30 章）

### 第一阶段 · Python 基础（本课程主讲教师负责）

1. Python、PyCharm 的安装与相关配置
2. 了解 Python，并编写第一个程序，常见的 bug
3. Debug、注释与输出函数
4. 变量与标识符
5. 数值类型、字符串与格式化输出
6. 算术与赋值运算符、输入函数与转义字符
7. if 判断、比较运算符与逻辑运算符
8. if-else、if-elif 与 if 嵌套
9. while 循环与循环嵌套
10. for 循环与 break、continue 关键字
11. 课上作业

### 第二阶段 · Python 进阶

12. 字符串编码与字符串常见操作
13. 字符串的查找、判断、修改
14. 列表的定义与增删改查，列表的推导式与嵌套
15. 元组、字典的定义与字典的增删改查
16. 字典的常见操作二、集合的格式及使用
17. 类型转换：int、float、str、eval、tuple、list、chr()
18. 赋值、深浅拷贝、可变与不可变对象
19. 函数、return 返回值与形参实参
20. 作用域、匿名函数与匿名函数的参数

### 第三阶段 · AI 编程实践

21. AI 核心概念讲解：LLM 到 Agent、Skill
22. 利用 Agent 进行编程（1）
23. 课上作业
24. 利用 Agent 进行编程（2）
25. 课上作业
26. AI 方法概述
27. 利用 Agent 进行 AI 方法构建（1）
28. 课上作业
29. 利用 Agent 进行 AI 方法构建（2）
30. 课上作业
