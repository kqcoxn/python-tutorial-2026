再来点循环：for、range()、break、continue
固定次数、提前结束、跳过本轮
WHY · 两类重复问题
重复次数已知，还是只知道条件？
01“输出 5 次” -> 次数已知
02“输入到合法为止” -> 条件控制
前者更适合 for，后者更适合 while。
依次取得一组整数
for number in range(5):
print(number)
输出：0 · 1 · 2 · 3 · 4
number 每一轮自动取得下一个值。
只写终点时，从 0 开始
range(5)
开始：0　终点边界：5　步长：1
实际序列：0、1、2、3、4
指定起点和终点边界
for number in range(2, 6):
print(number)
实际序列：2、3、4、5
包含起点，不包含终点
range(1, 5) -> 1、2、3、4
想输出 1 到 5：range(1, 6)
第三个参数决定每次走多远
for number in range(10, 3, -2):
print(number)
实际序列：10、8、6、4
写出全部实际值
range(3)
range(3, 7)
range(8, 1, -3)
揭示：0,1,2 ／ 3,4,5,6 ／ 8,5,2
点击揭示 · 结果
用最直接的结构表达题意
固定次数、整数范围 -> for
条件控制 -> while
两者有时都能完成，但清晰度不同。
条件满足，立即结束循环
for attempt in range(1, 4):
answer = input(f"第 {attempt} 次尝试：")
if answer == "python":
print("回答正确")
break
break 结束的是最近一层循环
它不会关闭 Python 程序。
循环后的代码仍然会继续执行。
跳过本轮剩余代码，进入下一轮
for number in range(1, 8):
if number % 2 == 0:
continue
print(number)
输出：1、3、5、7
KEY · 一句话区分
break 与 continue
break：后面的轮次也不要了
continue：只放弃本轮剩余部分
必要更新被跳过，就会死循环
count = 0
while count < 5:
if count == 2:
continue
count += 1
揭示：到 2 后，count += 1 永远执行不到。
点击揭示 · 答案
只退出内层，外层仍继续
for row in range(1, 4):
for column in range(1, 5):
if column == 3:
break
print(row, column)
每一行只输出第 1、2 列。
看到 break / continue，问三句
1. 它位于哪一层循环？
2. 哪些代码会被跳过？
3. 循环后下一行在哪里？
