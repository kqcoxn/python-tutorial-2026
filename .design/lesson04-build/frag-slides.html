单引号和双引号，都能创建字符串
line_a = "Python says: 'Hello!'"
line_b = '老师说："先预测，再运行。"'
print(line_a)
print(line_b)
外层用一种，内容里可以直接出现另一种。
字符串从哪开始、到哪结束？
message = "Hello"
英文直引号成对出现："..." 或 '...'
中文弯引号 “...” 不能代替代码引号。
+ 可以连接两个字符串
first_name = "Lin"
last_name = "Mu"
print(first_name + " " + last_name)
中间的空格来自字符串 " "，不是自动生成。
先猜：哪一个输出有空格？
print("Hello" + "Python")
print("Hello " + "Python")
揭示：HelloPython / Hello Python
点击揭示
这里为什么报错？
score = 96
print("得分：" + score)
揭示：字符串与整数不能直接用 + 连接。
更清楚的写法：f-string。
点击揭示 · 原因
把变量直接放进一句话
name = "小林"
score = 96
print(f"{name} 本次得分：{score}")
f 开启格式化；{} 中的值会被计算并显示。
三个部分，一个都不能少
print(f"姓名：{name}")
①前缀 f
②成对的字符串引号
③成对的 {} 占位表达式
先预测：输出是什么？
name = "小林"
print("欢迎，{name}")
揭示：输出 欢迎，{name}。花括号只是普通文字。
点击揭示
一句话，可以有多个占位表达式
name = "小林"
score = 96
rank = 2
print(f"{name}：{score} 分，第 {rank} 名")
ESCAPE · 转义字符
有些字符，键盘上看得见却不好直接写
反斜杠 \ 发出一个特殊信号：
\n：换行
\t：制表间隔
\\：一个反斜杠
一次 print()，输出三行
main.py — python3
print("第一行\n第二行\n第三行")
$ python main.py
第一行
第二行
第三行
\n 不会原样显示；它让后面的文字另起一行。
间隔与反斜杠
print("姓名\t分数")
print("C:\\Users\\student")
\t 只提供制表间隔，不保证复杂表格严格对齐。
外层引号冲突时，可以转义
print("他说：\"Hello!\"")
print('It\'s Python.')
\" 输出双引号；\' 输出单引号。
f-string 与换行一起用
name = "小林"
score = 96
rank = 2
print(f"姓名：{name}\n分数：{score}\n名次：{rank}")
字符串出错，按边界检查
1.引号是否英文、成对
2.f-string 前是否有 f
3.花括号是否成对
4.+ 两侧是否都是字符串
